Ссылка на репозиторий с проектом: https://github.com/Alexnov33X/Aton.git
Перед запуском проекта не забудьте установить все необходимые зависимости,
 проект запускался в Visual Studio 2022.
В программе остались куски кода от авторизации по JWT токенам,
я решил на полпути что обойдусь без них, ибо требования сложно интерпретировать
 с участием JWT токенов и авторизации.

Изначальный аккаунт администратора:
Login = "Admin",
Password = "Admin",
Birthday = DateTime.Now,
Name = "Admin",
CreatedBy = "Admin",
CreatedOn = DateTime.Now,
Admin = true

В методах параметры yourLogin и yourPassword - это логин и пароль пользователя инициатора запроса,
по умолчанию они задействуют данные начального администратора.
